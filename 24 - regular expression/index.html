regular expressions

  1. .source: 정규식 표현에서 // 안에 들어있는 것만 출력함
  2. .exec(): 정규식표현과 매치되는게 있으면 배열로 리턴, 없으면 null
  3. .test(): 매치되는게 있으면 true, 없으면 false
  4. flags... 정규표현식에 추가되는 조건?표현들은 // 뒤에 붙인다. ex) i(case-insensitive), g(나오는 족족 찾음), ...
  5. .match(): 검사할 문자열에 .match()를 사용하고 파라미터로 정규표현식을 전달한다. 매치되면 배열을 리턴, 없으면 null
  6. .search(): 매치되는 것의 인덱스를 반환. 없으면 -1
  7. .replace(): 첫번쨰 파라미터로 전달된 정규식 표현과 일치하는 것을 찾아서, 두번째 파라미터로 전달된 교체 문자와 바꾼다.
